# MEDICA_35735

## System obsługi pacjentów dla sieci klinik MEDICA+

Projekt został wykonany w celu informatyzacji procesu obsługi pacjentów w sieci klinik MEDICA+.

## Zakres projektu

- rejestracja pacjentów,
- zarządzanie wizytami,
- przechowywanie dokumentacji medycznej,
- obsługa lekarzy i harmonogramów,
- baza danych pacjentów,
- projekt UX/UI,
- integracja GitHub z Google Colab.

## Technologie

- Git
- GitHub
- SQL
- Google Colab
- Figma
